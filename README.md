Web Scraping using Python
Project Description
This project demonstrates basic web scraping using Python. The program collects quotes from the website https://quotes.toscrape.com and stores them in a CSV file.

Technologies Used
Python
Requests Library
BeautifulSoup (bs4)
CSV Module
How the Project Works
The Python program sends a request to the website.
It downloads the HTML content of the page.
BeautifulSoup parses the HTML and extracts the quotes.
The scraped quotes are saved into a CSV file.
Output
The extracted quotes are saved in a file named quotes.csv.

How to Run the Project
Install required libraries pip install requests beautifulsoup4

Run the Python file python web_scraping.py
